# Midi2Usb 1.6

Receives midi data from conventional 5pin and sends that out via USB

Needs libraries:
- midi.h
- Usb.h
- usbh_midi.h
- LiquidCrystal.h

## Hardware:
Needed for this project:
- Arduino
- Midi shield 
- USB Host
- LCD display (here 16x2 is used)

The LCD is not necessary to run the converter, but it is a nice addon to the box.
